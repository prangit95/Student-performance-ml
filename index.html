About This Project

End-to-end machine learning pipeline for predicting academic outcomes.

⬡

Pipeline

Data generation → cleaning → feature engineering → GridSearchCV tuning → evaluation. Run end-to-end with python main.py.

◈

Models

Six classifiers compete: Random Forest, Gradient Boosting, SVM, KNN, Decision Tree, Logistic Regression. Best model is saved automatically.

◇

Features

15 raw input features + 4 engineered features: study efficiency, sleep optimality, effort score, and parental support index.

○

Stack

Python · scikit-learn · pandas · NumPy · matplotlib · joblib. Frontend: vanilla HTML/CSS/JS + Chart.js.
